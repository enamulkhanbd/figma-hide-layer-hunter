Simple Steps to a Cleaner File

To build confidence and show how easy the plugin is to use, this section visually breaks down the process. It shows the simple, interactive steps required to use the plugin, demonstrating its straightforward and user-friendly design.

1

Define Your Scope

Select a specific frame or group to search within, or select nothing to scan the entire page.

2

Run the Plugin

Open your Plugins menu and run "Hide Layer Hunter". It instantly finds and selects all matching hidden layers.

3

Review & Delete

All found layers are now selected. Simply inspect them in the layers panel and press the 'Delete' key to clean up.
